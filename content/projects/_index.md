### [uniapp-video-player](https://github.com/liusheng22/uniapp-video-player) ⭐ 50+

**突破 uni-app 原生 video 层级限制的视频播放器**

一款突破 uni-app 原生 video 组件层级限制的视频播放器，兼容 iOS/Android 系统。通过 renderjs 使用 HTML5 原生 video 标签实现，可以被其他元素覆盖、遮挡，提供更高的页面定制性。支持 Vue2/Vue3 双版本，完美兼容 swiper/scroll-view。

**相关链接：** [GitHub 仓库](https://github.com/liusheng22/uniapp-video-player) ⭐ 43 | [DCloud 插件市场](https://ext.dcloud.net.cn/plugin?id=19654) ⭐ 82 | [npm 包](https://www.npmjs.com/package/uniapp-video-player)

### [Nonsense UI](https://nonsense-ui.lius.me/)

**制造交互障碍的 Vue 3 荒诞组件实验室**

一个专注于制造交互障碍的 Vue 3 整蛊组件库。拒绝用户体验，挑战直觉极限，提供反人类、反直觉的荒诞交互。虽然交互很荒诞，但代码实现非常严谨，使用 TypeScript 和 Vue 3 Composition API 开发。

**相关链接：** [官方文档](https://nonsense-ui.lius.me/) | [GitHub 仓库](https://github.com/liusheng22/nonsense-ui) | [npm 包](https://www.npmjs.com/package/nonsense-ui)

### [Flyio.js](https://flyio-js.vercel.app/doc/flyio/readme)

**基于 Promise 的 HTTP 请求库**

Flyio.js 是一个轻量级、功能强大的 HTTP 请求库，支持浏览器和 Node.js 环境。提供简洁的 API 设计，支持请求/响应拦截器、自动转换 JSON、请求取消等特性，让网络请求变得更加简单高效。
